3 cin

3.1 简介（参考链接）

cin 是 C++ 的标准输入流对象，即 istream 类的对象。cin 主要用于从标准输入读取数据，这里的标准输入，指的是终端的键盘。此外，cout 是输出流对象，即 ostream 类的对象。这里的标准输出指的是终端屏幕。

在理解 cin 功能时，不得不提标准输入缓冲区。当我们从键盘输入字符串的时候$\color{red}{需要敲一下回车键才能够将这个字符串送入到缓冲区中}$，那么敲入的这个回车键（\r）会被转换为一个换行符（\n），这个换行符也会被存储在 cin 的缓冲区中并且被当成一个字符来计算！比如我们在键盘上敲下了 123456 这个字符串，然后敲一下回车键（\r）将这个字符串送入了缓冲区中，那么此时缓冲区中的字节个数是 7 ，而不是 6。

4 sort

sort()定义在头文件中，接受两个或三个参数。当输入两个参数时，默认升序排序。若指定第三个参数则按照第三个参数指定的比较方式进行排序。

注意到代码中compare函数中“a>b”，对应的时降序。此处，不加考证，直观的假设为，sort函数的排序是基于小于运算进行升序排序的。当指定了第三个参数时，再排序时对小于运算符进行了重载。如果将小于运算重载为大于运算后，再按照此时的“小于运算”进行排序的得到的就是降序。

此外sort排序可以用于更加复杂的排序，如自定义的结构体的排序。
