它代表创建（Create）、更新（Update）、读取（Retrieve）和删除（Delete）操作。

CURD 定义了用于处理数据的基本原子操作。

CURD 操作通常是使用关系型数据库系统中的结构化查询语言（Structured Query Language，SQL）完成的。随着 Web 变得更加具有面向数据特性，因此需要从基于 SQL 的 CURD 操作转移到基于语义 Web 的 CURD 操作。
